# Security Policy

Use this section to tell people about which versions of your project are
currently being supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| >= 1.0  | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

Please do not open an issue for security vulnerabilities. Instead, please report any security vulnerabilities directly to the maintainers, for example by emailing them. We will acknowledge receipt of your vulnerability report and strive to send you regular updates about our progress. If the vulnerability is accepted, we will implement a fix, and communicate the release details to you.
